2. ğŸ—ï¸ Struktur Dasar HTML5

Setiap dokumen HTML5 harus memiliki struktur dasar yang konsisten:

<!-- Deklarasi dokumen HTML5 --> <!DOCTYPE html> <!-- Elemen root/utama --> <html lang="id"> <!-- Bagian HEAD - tidak terlihat di browser --> <head> <!-- Encoding karakter --> <meta charset="UTF-8"> <!-- Viewport untuk responsive design --> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- Judul halaman (muncul di tab browser) --> <title>Judul Halaman</title> </head> <!-- Bagian BODY - konten yang terlihat --> <body> <!-- Konten website di sini --> <h1>Hello World!</h1> </body> </html>

ğŸ“ Penjelasan Setiap Bagian:

1. DOCTYPE Declaration

<!DOCTYPE html>
  • Memberitahu browser bahwa ini adalah dokumen HTML5
  • Harus selalu ada di baris pertama
  • Tidak case-sensitive, tapi konvensi menggunakan lowercase

2. HTML Element

<html lang="id">
  • lang="id" menandakan bahasa Indonesia
  • Membantu screen reader dan SEO
  • Contoh lain: lang="en" untuk bahasa Inggris

3. Head Element

Berisi metadata yang tidak terlihat di halaman web:

  • charset="UTF-8": Encoding untuk karakter internasional
  • viewport: Pengaturan untuk responsive design
  • title: Judul yang muncul di tab browser

4. Body Element

Berisi semua konten yang terlihat di browser

Tips: Selalu gunakan indentasi (spasi/tab) untuk membuat kode lebih mudah dibaca!

ğŸ“‹ Daftar Semantic Elements:

  • <main>: Konten utama (hanya 1 per halaman)
